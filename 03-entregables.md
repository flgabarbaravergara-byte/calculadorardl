# Entregables

Al finalizar el proyecto se esperan los siguientes elementos como evidencia del trabajo realizado:

- Código fuente de la aplicación, de ser posible en un repositorio (GitHub, ZIP o similar).
- Un archivo `README.md` dentro de la carpeta del código fuente explicando la funcionalidad.
- Capturas de pantalla mostrando la aplicación en funcionamiento.
- Breve documentación y análisis del proyecto.

Todo esto se comparte a través de la plataforma de aprendizaje.

## Portafolio

Se recomienda subir el proyecto a GitHub e incluirlo en el portafolio de desarrollo web. Una buena presentación en GitHub Pages, CodePen o similar ayuda a demostrar las habilidades en JavaScript.
