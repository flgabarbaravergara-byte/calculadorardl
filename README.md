# Calculadora de Evaluación Fonoaudiológica — Módulo 3

Proyecto final del **Módulo 3: Fundamentos de programación en JavaScript** (curso Aalkemy).

## Objetivo del proyecto

Desarrollar una aplicación en consola que permita realizar operaciones matemáticas básicas, usar estructuras condicionales y de bucles, modularizar el código con funciones, y trabajar con arreglos y objetos — todo aplicado a un caso de uso real: el seguimiento de pacientes en evaluaciones de fonoaudiología.

A lo largo de 5 lecciones, el proyecto va construyendo de forma incremental una pequeña "calculadora" orientada a fonoaudiología, donde cada ejercicio introduce un concepto nuevo de JavaScript sobre la misma temática.

## Estructura del repositorio

- `ejercicio 1/` — Introducción al lenguaje JavaScript (`console.log`, `prompt`, `alert`).
- `ejercicio 2/` — Variables, expresiones y sentencias condicionales (`if`, `switch`, operaciones matemáticas).
- `ejercicio 3/` — Arreglos y ciclos (`for`, `while`, funciones de filtrado).
- `ejercicio 4/` — Funciones (modularización, funciones que reciben parámetros y retornan resultados, funciones llamadas dentro de otras funciones).
- `ejercicio 5/` — Objetos (propiedades, métodos con `this`, arreglos de objetos recorridos con `forEach()` y `map()`).

Cada carpeta de ejercicio incluye:

- El archivo `index*.html` para ejecutar el script en el navegador.
- El archivo `leccion*.js` con el código.
- Un `README.md` propio con la explicación detallada del ejercicio y las capturas de pantalla como evidencia.

## Cómo ejecutar cualquier ejercicio

1. Entrar a la carpeta del ejercicio correspondiente.
2. Abrir el archivo `index*.html` en el navegador.
3. Abrir la consola del navegador para ver el resultado:
   - Chrome (Mac): `⌘ + Option + J`
   - Safari (Mac): habilitar el menú Desarrollador y usar `⌘ + Option + C`
